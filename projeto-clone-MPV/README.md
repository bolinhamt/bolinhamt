Projeto SM5 - Do Clone ao Produto Mínimo Viável (MVP+)
📝 Descrição do Projeto
Neste projeto, conduzi a evolução de um clone funcional para um MVP+ com diferenciais claros de produto e justificativas técnicas de priorização.

A análise combinou engenharia reversa, prompting e decisões de UX para transformar cópia inicial em proposta com identidade própria.

🧰 Tecnologias Utilizadas
HTML5 CSS3 JavaScript AI_Studio

Base técnica: FluxoMD
Apoio de IA: prompts no AI Studio
Objetivo: diferenciação funcional e ganho de usabilidade
📊 Resultados e Aprendizados
4 diferenciais priorizados: histórico local, pin de arquivos, buscar/substituir por modal e estatísticas em tempo real.
Decisão técnica: adotei diferenciais de alto impacto em fluxo de edição para maximizar valor com escopo controlado.
Aprendizado analítico: MVP+ exige balancear tempo de implementação, custo cognitivo da interface e retorno prático ao usuário.
